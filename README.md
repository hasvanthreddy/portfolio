# Portfolio Website

A personal portfolio website showcasing my projects, skills, and achievements.  
This site is built to provide a professional online presence and make it easy for recruiters, collaborators, and peers to explore my work.

---

## Features
- Home page with introduction and tagline
- About section with background and interests
- Projects showcase with descriptions and links
- Skills section highlighting technologies and tools
- Contact section with email and social links

---

## Tech Stack
- HTML5 for structure
- CSS3 for styling and layout
- JavaScript for interactivity

## How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/hasvanthreddy/portfolio.git
   cd portfolio
2. Open the project in VS Code (or any editor).

3. Right-click index.html and select Open with Live Server (requires the Live Server extension).

4. The site will open in your browser at:
   http://127.0.0.1:5500/index.html
